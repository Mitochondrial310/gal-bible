# 📖 ギャル聖書アプリ

聖書をギャル語で楽しく学べるWebアプリ💅✨

## 🚀 セットアップ手順

### 1. Node.jsをインストール
https://nodejs.org から LTS版をダウンロード＆インストール

### 2. このフォルダをターミナルで開く
```bash
cd gal-bible
```

### 3. ライブラリをインストール
```bash
npm install
```

### 4. 開発サーバーを起動
```bash
npm run dev
```
ブラウザで http://localhost:5173 を開いたら完成！

---

## 🌐 Vercelで公開する手順

1. GitHubにこのフォルダをアップロード
2. https://vercel.com にアクセスしてGitHubでログイン
3. 「New Project」→ リポジトリを選択
4. 「Deploy」をクリック
5. 自動でURLが発行される！
